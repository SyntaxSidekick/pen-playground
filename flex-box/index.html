Nested Flexbox Layout

Nested Container 1

A
B
C

Nested Container 2

X
Y
Z

HTML

<div class="flex-container nested-parent">
    <div class="flex-item nested-child">
        <h4>Nested Container 1</h4>
        <div class="inner-flex">
            <div class="inner-item">A</div>
            <div class="inner-item">B</div>
            <div class="inner-item">C</div>
        </div>
    </div>
</div>

CSS

.nested-parent {
    gap: 2rem;
}

.nested-child {
    flex: 1;
    padding: 1rem;
}

.inner-flex {
    display: flex;
    gap: 0.5rem;
    
    &.column {
        flex-direction: column;
    }
}

Card Layout Example

Card Title 1

New

This is some content for the first card.

2 hours ago

Card Title 2

Featured

This is content for the second card with more text to show flexibility.

5 hours ago

Card Title 3

Third card content.

1 day ago

HTML

<div class="card-container">
    <div class="card">
        <div class="card-header">
            <h3>Card Title 1</h3>
            <span class="card-badge">New</span>
        </div>
        <div class="card-content">
            <p>This is some content for the first card.</p>
        </div>
        <div class="card-footer">
            <button class="btn">Action</button>
            <span class="card-meta">2 hours ago</span>
        </div>
    </div>
</div>

CSS

.card-container {
    display: flex;
    gap: 1.5rem;
    flex-wrap: wrap;
}

.card {
    display: flex;
    flex-direction: column;
    flex: 1;
    min-width: 280px;
    
    .card-header {
        display: flex;
        justify-content: space-between;
        align-items: center;
    }
    
    .card-footer {
        display: flex;
        justify-content: space-between;
        align-items: center;
        margin-top: auto;
    }
}
